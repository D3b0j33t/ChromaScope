## Getting Started

1. Clone this repository:

   ```bash
   git clone https://github.com/D3b0j33t/ChromaScope.git
   ```

2. Open `index.html` in your web browser.

That's it! No additional setup or dependencies required.

## Usage

1. The color swatch will display the current color
2. Use the interface to:
   - View color name
   - Copy color values in different formats
   - See color variations and alternatives
